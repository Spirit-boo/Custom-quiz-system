# 执行步骤 (PWA 网页应用)

## 开发原则
- **小步推进**：每步完成一个独立可验证的模块
- **先数据后界面**：先搭建数据层（js/），再做页面视图
- **一屏一验收**：每个视图完成后用浏览器测试
- **及时记录**：每天更新 dev-logs 日志
- **用户可随时用**：每一步完成后都可以在浏览器中打开看到效果
