# 开发需求文档

## 项目概述
个人定制刷题小程序，基于微信小程序平台，纯本地存储，支持手机和电脑微信内使用。

## 核心功能需求

### F1. 题库导入
- **支持格式**: Excel (.xlsx / .xls)、CSV (.csv)、TXT (.txt)
- **标准字段**: 题目类型、题目内容、选项A-G（2-7个）、正确答案、解析、标签（可选）
- **字段映射**: 用户导入时手动将文件列映射到标准字段
- **数据校验**:
  - 题目内容不能为空
  - 答案不能缺失
  - 题型必须为: single / judge / multiple / fill
  - 选项数量需与题型匹配
  - 答案必须在选项范围内
  - 检测完全重复的题目
- **校验反馈**: 通过/警告/错误三级，错误项支持查看和修改

### F2. 题目搜索
- 模糊搜索：题目文本、解析、标签
- 结果高亮关键词
- 点击查看完整题目详情

### F3. 答题设置
- 题目数量：10 / 20 / 50 / 全部 / 自定义
- 出题模式：随机出题 / 顺序出题 / 错题重做 / 收藏专练
- 题型筛选：单选 / 判断 / 多选 / 填空，可多选组合
- 题库统计：显示各题型当前数量

### F4. 答题流程
- 逐题显示，一次一题
- 顶部进度条（当前题号/总题数）
- 不同类型交互：
  - 单选/判断：点击选项
  - 多选：多选+确认按钮
  - 填空：输入框+确认按钮
- 提交后显示：对错标记、正确答案、解析
- 收藏按钮（答题中可收藏）

### F5. 答题报告
- 总分与正确率
- 各题型正确率统计
- 用时统计
- 错题列表（可展开查看详情）
- 正确率趋势图（有历史记录时）

### F6. 错题本
- 错题自动收录
- 错误次数记录
- 可手动移除
- 支持重做错题

### F7. 收藏夹
- 答题时手动收藏
- 列表管理、取消收藏
- 支持练习收藏题目

### F8. 进度保存
- 答题中自动保存（每题提交后）
- 退出时提醒保存
- 下次进入可恢复未完成答题

## 非功能需求
- 纯本地存储，无需登录/注册
- 响应式设计，适配手机和电脑微信窗口
- UI 简洁直观，淡蓝色主题
- 操作响应流畅，无明显卡顿
- 微信小程序包体积 < 2MB
