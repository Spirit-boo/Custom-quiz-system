# 设计规范

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#4A90D9` | 按钮、选中态、标题强调 |
| 主色浅 | `#7AB3E8` | 悬停态、次要高亮 |
| 主色淡 | `#D6E9F8` | 卡片背景、选中背景 |
| 背景色 | `#F0F6FB` | 页面全局背景 |
| 卡片背景 | `#FFFFFF` | 卡片、列表项背景 |
| 文字主色 | `#333333` | 标题、正文 |
| 文字次要 | `#666666` | 辅助信息、日期 |
| 文字提示 | `#999999` | 占位符、禁用态 |
| 正确色 | `#52C41A` | 答对标记 |
| 错误色 | `#FF4D4F` | 答错标记 |
| 警告色 | `#FAAD14` | 警告提示 |
| 边框色 | `#E8E8E8` | 分割线、卡片边框 |

## 排版

| 层级 | 字号 | 用途 |
|------|------|------|
| H1 | 40rpx | 页面标题 |
| H2 | 36rpx | 区块标题 |
| H3 | 32rpx | 卡片标题 |
| 正文 | 28rpx | 题目、选项、解析 |
| 辅助 | 24rpx | 标签、日期、统计 |
| 小字 | 20rpx | 提示、脚注 |

## 间距

| 级别 | 值 | 用途 |
|------|-----|------|
| XS | 8rpx | 紧凑间距 |
| S | 16rpx | 元素内间距 |
| M | 24rpx | 卡片内间距 |
| L | 32rpx | 模块间距 |
| XL | 48rpx | 页面边距、大分隔 |

## 圆角与阴影

- 卡片圆角: `16rpx`
- 按钮圆角: `12rpx`
- 标签圆角: `8rpx`
- 卡片阴影: `0 2rpx 12rpx rgba(0,0,0,0.08)`
- 按钮阴影: `0 4rpx 12rpx rgba(74,144,217,0.3)`

## 组件风格

### 按钮
- 主按钮: 主色背景 + 白字 + 圆角 + 阴影
- 次按钮: 白底 + 主色边框 + 主色文字
- 文字按钮: 无边框无背景，主色文字
- 禁用态: 灰色背景 + 浅色文字

### 卡片
- 白底 + 圆角 + 轻阴影
- 内边距统一 24rpx

### 题型标签
- 单选: 蓝色 `#4A90D9`
- 判断: 绿色 `#52C41A`
- 多选: 橙色 `#FA8C16`
- 填空: 紫色 `#722ED1`

### 进度条
- 背景: `#E8E8E8`
- 填充: `#4A90D9`
- 高度: `8rpx`
- 圆角: `4rpx`

## 页面布局规范

- 页面最大宽度: `750rpx`（小程序标准屏宽）
- 内容区留白: 左右各 `32rpx`
- 页面底部留出安全距离（iPhone 底部横条兼容）
- TabBar 页面使用 `page-container` 确保不被遮挡

## 交互规范

- 点击区域最小: `80rpx × 80rpx`（防止误触）
- 按钮点击反馈: 透明度变化 `opacity: 0.8`
- 卡片点击反馈: 缩放 `scale(0.98)`
- 页面过渡: 微信默认右滑进入
- 加载状态: 居中转圈动画
- 空状态: 图标 + 提示文字
